# MCF-Crosstalk-Simulator

マルチコアファイバ(MCF: Multi-Core Fiber)のコア間クロストークをMATLABでシミュレーションするコード集です。大学卒業研究で使用しています。

解析手法として、結合モード理論(Coupled Mode Theory)による解析解アプローチと、有限要素法(FEM, MATLAB PDE Toolbox)によるスーパーモード解析アプローチの両方を含みます。

## ファイル一覧

### 結合モード理論(CMT)ベース

- `test.m` — 2コア間のクロストークを、孤立コアの伝搬定数(Bessel関数による特性方程式)と結合係数から解析的に計算する方向性結合器モデル。
- `multicore.m` — 上記をNコアに拡張し、断面図でコアをドラッグして配置を変えながらクロストークを確認できるGUI版。

### FEM(有限要素法)ベース

- `keny.m` — PDE ToolboxでコアのスーパーモードをFEM解析し、ドラッグ操作で断面配置を変えながらクロストークを計算する初期版。
- `multicore_FEM.m` — 上記のn_clad/n_coreを可変入力にした版。
- `waveleght.m` — FEM版をベースに、ファイバ長を短く(1m)して伝搬に伴うパワーのうなり(ビート)が見えるようにした版。
- `MCF_XT.m` — 最新版。クラッド径を125μm固定・ドラッグ不可にし、コア半径を1本ずつ入力可能にした上で、各コアのクロストークに加えて、コアごとのパワー+残りクラッド部のパワーの合計が1に保たれるかをエネルギー保存則で検証するグラフを追加している。

### `don't use/`

課題(kadai)用に作成した過去の試作スクリプト群(結合電力理論(CPT) + 自己相関関数による曲げ起因クロストークモデルなど)。参考として残しているが、本編の解析には使用していない。

## 主な物理モデル

- 弱導波近似のスカラー波動方程式: `-∇²ψ - k0²n(x,y)²ψ = λψ` (λ = -β²)
- ガイドモード条件: `n_clad·k0 < β < n_core·k0`
- スーパーモード展開によるコア間パワー結合、クロストーク `XT_k(z) = 10log10(P_k(z)/P_1(0))` [dB]

## 動作環境

MATLAB + PDE Toolbox (FEM系ファイルで使用)
